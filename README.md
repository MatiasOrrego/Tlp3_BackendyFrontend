# Buenas

Este proyecto es una aplicación web simple que permite al usuario ingresar su nombre y recibir un saludo personalizado desde un servidor backend.

## Tecnologías Utilizadas

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js con Express
- **Estilos**: Tema oscuro con diseño responsivo

## Estructura del Proyecto

## Cómo Ejecutar el Proyecto

### 1. Clonar el Repositorio
```bash
- "git clone https://github.com/MatiasOrrego/Tlp3_BackendyFrontend.git"
- "cd Tlp3_BackendyFrontend"

### 2. Instalar dependencias
npm install

### 3. Iniciar Servidor
- "node server.js"

### 4. Abrir el Frontend
"Abre el archivo index.html en tu navegador o utiliza una extensión como Live Server en Visual Studio Code para servir el archivo."

# Funcionalidades
1. Ingresar Nombre: El usuario puede ingresar su nombre en el campo de texto.
2. Recibir Saludo: Al hacer clic en el botón "Enviar", el servidor responde con un saludo personalizado.
3. Tema Oscuro: La página tiene un diseño moderno con tema oscuro.

#Ejemplo de Uso
1. Ingresa tu nombre en el campo de texto.
2. Haz clic en el botón "Enviar".
3. Verás un saludo personalizado en la página y en una alerta emergente.
